# Features Specification

## 1. AI Greeting Cards
**Spec:**
- Users select a theme (Deep Religious, Casual Family, Professional).
- Input recipient name and relationship.
- AI generates a custom image + poetic text using Fal.ai/OpenRouter.
- **Monetization:** Free watermarked download. $1 for HD/No-Watermark.

## 2. AI Meal Planning (Sahur & Iftar)
**Spec:**
- User inputs dietary prefs, family size, budget.
- Generates 7-day or 30-day plan.
- **Monetization:** Free 3-day plan. Subscription for full Ramadan plan + Shopping List.

## 3. AI Gift & Outfit Recommendations
**Spec:**
- "What do I buy for my mother-in-law?"
- "Matching outfits for family of 4."
- **Monetization:** Affiliate links (Shopee, Tokopedia, Amazon).

## 4. User System
- Login via Email/Google.
- Dashboard for saved cards/plans.

## 5. Admin Dashboard
- View usage stats.
- Manage content templates.
